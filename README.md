## 몬스터
  - **일반**
    - **작은 물고기**: 플레이어와 동일한 y좌표에 있을 경우 앞으로 돌진합니다.
    - **큰 물고기**: 정해진 범위를 돌아다닙니다.
    - **꽃게**: 플레이어를 발견하면 빠른 속도로 플레이어를 쫓아가며, 스킬 사용 시 플레이어에게 1초마다 반 칸씩 3번의 출혈 데미지를 줍니다.
  - **보스 (자라)**
    - 스킬
      - 평타(내려찍기): 플레이어가 특정 범위 내에 들어올 경우 자라가 제자리에서 점프하여 플레이어에게 한 칸의 피해를 준다.
      - 물대포: 위에서 수많은 물방울을 떨어뜨려 플레이어에게 반 칸의 피해를 준다.
      - 돌진: 돌진 준비 후에 필드의 랜덤 위치에서 자라가 돌진을 한다. 플레이어에게 두 칸의 피해를 준다.
